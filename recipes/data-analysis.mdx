## Overview

The **Data Analysis Agent** queries databases, generates reports, and shares insights with your team by connecting data sources, documentation tools, and communication platforms.

<Check>
  **Result:** Reduce time to insight from hours to minutes, automate reporting
</Check>

## Setup

<Steps>
  <Step title="Connect Services">
    - **Supabase** (OAuth): Select project and configure database access
    - **Notion** (OAuth): Select workspace
    - **Slack** (OAuth): Select workspace and channels
  </Step>
  <Step title="Deploy">
    Click Deploy to activate your Data Analysis Agent
  </Step>
</Steps>

## Usage

**Example prompts:**
- "Generate this week's user signup report"
- "How many premium users signed up this month?"
- "Show me daily active users for the last 7 days"

## Key Features

- **SQL Queries**: Execute queries on Supabase database
- **Report Generation**: Create and update Notion pages
- **Team Sharing**: Post insights to Slack channels
- **Data Exploration**: List tables and analyze database structure

## Best Practices

- **Use Read-Only Credentials**: Configure Supabase with read-only access to prevent accidental data modification
- **Test Queries First**: Validate SQL queries in Playground before deploying to production
- **Set Up Report Templates**: Create Notion database templates for consistent report formatting
