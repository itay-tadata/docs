<Update label="March 9 2026">
  ### New Features

  - **Claude Code** - Tadata agents now run on Claude Code under the hood. Agents are capable of bigger, more complex tasks — they'll stay on track with a to-do list, use per-conversation memory, and write scripts to bridge any gaps needed to accomplish their goals.
  - **Slack-style agent navigation** - Agents now live in the left sidebar as persistent collaborators, much like a Slack DM thread with a teammate you work with regularly.
  - **Agent identity in chat** - The chat sidebar now shows the agent's instructions and connectors while it runs, so you always have context on what your agent is working with.
  - **Delete connectors** - You can now remove connectors from an agent, making it easier to iterate on your setup without starting from scratch.

  ### Improvements

  - Better handling of large data responses from connectors such as Gmail and Fathom.
</Update>

<Update label="February 22 2026">
  ### New Features

  - **Templates** - Grab a pre-built template inside Tadata for common workflows: meeting prep, documenting feature requests from video calls, and identifying conference leads. We've also posted build-along demos to our [YouTube channel](https://www.youtube.com/@TadataHQ)
  - **Webhooks** - Tadata agents can now be triggered via external HTTP webhooks, allowing third-party services and applications to programmatically invoke agents with custom payloads.
  - **Duplicate agents** – Clone any agent, including its auth configuration.

  ### Connector Updates

  **Connector Improvements**
  - HubSpot: Now supports write operations

  **Newly Added Connectors**

  - *Notetakers* - Granola
  - *Databases* - Weaviate
  - *Sales & Lead Intelligence* - LinkedIn Post Comments, LinkedIn Post Reactions
</Update>

<Update label="February 12 2026">
  ### New Features

  - **Prompt to Create an Agent** - Describe what you want to accomplish and Tadata will automatically expand your prompt and suggest the right connectors to get the job done.
  - **Generate Missing Connectors** - Don't see a connector you need? Go to `Connectors` > `+ Add new connector` > `Generate with AI`. Then enter the connector name and Tadata will generate a connector for it in ~5 minutes. Keep the tab open while it works.

  ### Connector Updates

  **Connector Improvements**
  - Gmail: Added draft tool + improved handling of longer messages
  - Fathom: Better support for longer transcripts

  **Newly Added Connectors**

  - *Sales & Lead Intelligence* - Apollo.io, HeyReach, Hunter.io, Lemlist, LinkedIn, RocketReach, UpLead, Voila Norbert
  - *Forms & Surveys*  - Jotform, Typeform
  - *Communication* - Outlook Mail
  - *Security & Compliance* - CIRCL Vulnerability DB, CISA KEV, OpenSSF Scorecard
  - *Platform Integrations* - Mixpanel, Webflow

  ### Bug Fixes

  - Agent-creation progress now saves when you resize your browser window. No losing progress with the message that Tadata is intended for desktop screens.
</Update>
